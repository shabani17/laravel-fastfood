# Laravel FastFood

A full-stack FastFood ordering system built with Laravel.  
This repository contains both the **Main User Project** and the **Admin Panel** for managing orders, transactions, charts, and reports.

---

## Features

- User authentication & authorization
- Admin dashboard with monthly transaction chart (AmCharts)
- Order management (create, update, track)
- Transaction history
- Responsive UI
- RESTful architecture
- Persian calendar support (Verta)

---

## Project Structure

| Folder                 | Description                                    |
| ---------------------- | ---------------------------------------------- |
| `/laravel-app`         | Main user-facing project                       |
| `/laravel-admin-panel` | Admin panel for managing orders & transactions |
